Install Java 11

Download an appropriate JDK for your operating system. We will be using OpenJDK for this tutorial.

Once installed, you can use the java command from your command line.

Check if the correct java version is installed:


$ java -version
openjdk version "11" 2018-09-25
OpenJDK Runtime Environment 18.9 (build 11+28)
OpenJDK 64-Bit Server VM 18.9 (build 11+28, mixed mode)

You need to set the JAVA_HOME environment variable to a JDK 11 installation directory. You can follow this guide to set JAVA_HOME for your platform.

Note: If you have newer or older versions of Java installed along with JDK 11, you need to make sure that the JAVA_HOME environment variable points to JDK 11.
